# teknia-tools (TEKNIA pack)

Self-contained Node.js tooling + schemas + sample TEKTOKs/aggregations for TEKNIA.

## Quickstart (PowerShell)

npm i
npm run validate:tektok
npm run validate:agg
npm run nv

## JSON Schema export

npm run export:schemas

Outputs:

- schemas/json/tektok.schema.json
- schemas/json/teknia.aggregation.schema.json


## Examples

npm run check:examples
